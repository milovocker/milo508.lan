Vas a crear una pequeña aplicación web en PHP que simule el inicio de sesión de un usuario y el almacenamiento de su preferencia de idioma. La aplicación tendrá tres páginas: una página de inicio de sesión, una página de bienvenida y una página de configuración de idioma.

Paso 1: Página de Inicio de Sesión (login.php)
    Crea un formulario HTML que solicite al usuario su nombre y tenga un botón para enviar el formulario.
    Cuando el usuario envíe el formulario, guarda el nombre del usuario en una variable de sesión utilizando $_SESSION.
    Redirige al usuario a una página de bienvenida si el inicio de sesión es exitoso.

Paso 2: Página de Bienvenida (bienvenida.php)
    En esta página, muestra el nombre del usuario utilizando la variable de sesión $_SESSION['nombre_usuario'].
    Si el usuario no ha iniciado sesión, redirígelo de vuelta a la página de inicio de sesión.
    Añade un enlace que permita al usuario ir a la página de configuración de idioma.

Paso 3: Página de Configuración de Idioma (configuracion_idioma.php)
    Crea un formulario que permita al usuario seleccionar su idioma preferido (por ejemplo, "español", "inglés" o "francés").
    Al enviar el formulario, guarda la preferencia de idioma del usuario en una cookie que expire en 7 días.
    Redirige al usuario de vuelta a la página de bienvenida.

Paso 4: Modificar la Página de Bienvenida para Usar Cookies
    Modifica la página de bienvenida para que se muestre un mensaje personalizado basado en el idioma seleccionado por el usuario (almacenado en la cookie). Por ejemplo:
    Si el idioma es español: "Bienvenido, [nombre_usuario]".
    Si el idioma es inglés: "Welcome, [nombre_usuario]".
    Si el idioma es francés: "Bienvenue, [nombre_usuario]".
    Si no se ha seleccionado ningún idioma (la cookie no existe), muestra un mensaje genérico en español.
    
Paso 5: Cerrar Sesión (logout.php)
    Crea una página o botón de "Cerrar Sesión" que destruya la sesión y redirija al usuario de vuelta a la página de inicio de sesión.
    Asegúrate de que las cookies no se borren al cerrar la sesión, de modo que si el usuario inicia sesión de nuevo, su preferencia de idioma siga estando disponible.
    Entregables:
    Archivos PHP: login.php, bienvenida.php, configuracion_idioma.php, y logout.php.
    Una breve explicación del flujo de trabajo entre las páginas, el uso de variables de sesión y cookies, y cómo cada página interactúa con los datos almacenados.
